Detailed Description

Classe CondomiX representa a Condomix, a empresa especialista na gestão de condomínios, através de um contentor de objetos para armazenar os prestadores de serviço de limpeza e através de uma árvore binária de pesquisa, onde é guardada a informação dos condomínios que administra.
